# XX 自定义编码分析（卡牌名称表 lang=0 槽）

**写作日期**：2026-04-17
**前置依赖**：
- `data/card-names.s`（lang=0 槽）
- `refs/yugioh-card-search/data/card_master.csv`（日文卡名对照）
- `data/card-name-pointer-table.s`（指针表，确认每卡 XX 起止）

---

## 一、背景

游戏王 EX2006 (BY6E, 北美版) ROM 的卡名表 (`card_names_pool @ 0x015BB594`) 为每张卡保存 6 个 null 终止字符串：

```
slot 0: XX (lang_id=0)  —— 自定义编码 (本文研究目标)
slot 1: EN              —— ASCII
slot 2: DE
slot 3: FR
slot 4: IT
slot 5: ES
```

Wiki 反汇编 `0x080EE7AC` 注释 `language_id=0` = "Japanese"，但北美版用户无法通过设置切换到此语言。XX 字段在游戏运行时可能被用于卡名搜索/排序。

实证特征：
- **永远偶数长度** (6–32 B) → 每字符 2 字节定长
- 高字节几乎全在 `0xF0`–`0xFF`
- 字符对数 ≈ 日文卡名汉字+假名+全角符号总数 (`青眼の白龍` 5 字 → 5 对)

## 二、编码方案

XX 是 **2 字节定长编码**：每字符 = `<HI 字节>` `<LO 字节>`。

`HI` 字节决定字符类别：

| HI 范围 | 含义 | 已确认覆盖 |
|---|---|---|
| `0xF0` | 特殊符号 (・, ー, 数字, 括号等) | 部分 (・ ー) |
| `0xF1` | 平假名 (HI=F1, LO=80..CE) + 片假名前半 (HI=F1, LO=D0..FF) | **完全** |
| `0xF2` | 片假名后半 (LO=80..A1) + 其他 (LO=A2..FF, 疑似汉字) | 片假名完全 |
| `0xF3`..`0xFE` | 汉字 (sparse, 每字唯一编码) | 仅 ~10 个手工对齐 |
| `0xFF` | 极少出现 (5 个 lo-byte) | 未知 |

## 五、F0 特殊符号 + 全角字母

| LO 范围 | 含义 | 来源 |
|---|---|---|
| `0x80` | 全角空格 ` ` (高频，131 次) | cid=2000 "E·HERO スチーム..." 验证 |
| `0x84` | `・` 中点 | 多卡 |
| `0x8B` | `ー` 长音符 | 多卡 |
| `0xC8`..`0xE1` | **全角拉丁大写 A..Z** | cid=2000 "E·HERO" 解码 (E=CC, H=CF, R=D9, O=D6) |
| `0x9E`/`0xFF`/`0xCC`/`0xDD` 等 | 未知（疑似全角小写或数字） | 仍需对照扩充 |

**重要发现**：F0 row 不仅含特殊符号，还**完整包含全角拉丁字母**（`F0 C8`..`F0 E1` = `Ａ..Ｚ`）。
这意味着像 "E·HERO" 这种英文片段在 JP 名里是用全角字母而非 ASCII 表示的。

F0 row 共 48 个不同 lo-byte 出现，已识别 29 个（空格 + 中点 + 长音 + 26 字母）。
剩余 19 个 lo-byte（多在 0x80..0xC7 范围）疑似全角小写字母 / 数字 / 标点。

## 九、附：解码脚本骨架

```python
F1_KATA = "アィイゥウェエォオカガキギクグケゲコゴサザシジスズセゼソゾタダチヂッツヅテデトドナニヌネノハバパ"
F2_KATA = "ヒビピフブプヘベペホボポマミムメモャヤュユョヨラリルレロヮワヰヱヲン"
HIRA = {0x86:'か', 0x87:'が', 0x88:'き', ..., 0xCE:'ん'}
F0_SP = {0x84:'・', 0x8B:'ー'}

def decode_xx(xx_bytes):
    out = []
    for i in range(0, len(xx_bytes), 2):
        hi, lo = xx_bytes[i], xx_bytes[i+1]
        if hi == 0xF1 and lo >= 0xD0:
            out.append(F1_KATA[lo - 0xD0])
        elif hi == 0xF2 and lo <= 0xA1:
            out.append(F2_KATA[lo - 0x80])
        elif hi == 0xF1 and lo in HIRA:
            out.append(HIRA[lo])
        elif hi == 0xF0:
            out.append(F0_SP.get(lo, f'[F0_{lo:02X}]'))
        else:
            out.append(f'[{hi:02X}{lo:02X}]')   # 汉字占位
    return ''.join(out)
```
