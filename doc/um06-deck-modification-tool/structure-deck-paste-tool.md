此 Sheet 是交互工具，输入区为空时公式显示 `#N/A`，不代表真实数据。

## 列说明

| 列 | 说明 |
|----|------|
| A (行号) | 序号（最多 191 张）|
| B (Paste Below) | 粘贴卡牌密码 |
| C (Quantity) | 数量 |
| D (Name) | 自动查找的卡名 |
| E (Structure Decks) | 该卡在预组中的 Hex 编码 |
| F (0 Padder) | 补零后的 4 位 Hex（写入 ROM 用）|
